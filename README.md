# Vehicle Renting Project

This test project is an admin web application for renting vehicles.
It allows the Admin to browse available vehicles, make reservations, manage bookings and customers.
The application is built using FastAPI for the backend and SQLModel for database interactions.
The database used is MySQL, and the application supports features such as user authentication and
vehicle management.

# Stacks Used:

  - Python
  - JavaScript
  - HTML
  - CSS
